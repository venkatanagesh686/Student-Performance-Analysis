# Student Performance Analysis

## 📊 Project Overview

This project analyzes student performance data using **Python, Pandas, and Matplotlib**.
The goal of this project is to explore and visualize student exam scores in **Math, Reading, and Writing** to identify trends and insights.

---

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains information about student performance in exams.

Main columns include:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

---

## 📈 Analysis Performed

* Data loading using **Pandas**
* Data exploration and cleaning
* Average score calculation
* Visualization of **Math score distribution**
* Identifying patterns in student performance

---

## 📊 Visualization

The project includes visualizations such as:

* Math score distribution histogram
* Score comparisons
* Basic performance trends
